# Como Ativar o GitHub Pages

Para tornar a aplicação GrowUp Kids acessível online, siga estes passos:

## Passos para Ativar GitHub Pages

1. **Aceda ao seu repositório no GitHub:**
   - Vá para: https://github.com/MariaSambe/growup-kids

2. **Abra as Configurações:**
   - Clique na aba **Settings** (Configurações) no topo do repositório

3. **Navegue até Pages:**
   - No menu lateral esquerdo, clique em **Pages**

4. **Configure a Source (Origem):**
   - Em **Source**, selecione **Deploy from a branch**
   - Em **Branch**, selecione **main**
   - Em **Folder**, selecione **/ (root)**
   - Clique em **Save**

5. **Aguarde o Deployment:**
   - O GitHub levará alguns minutos para fazer o deployment
   - Quando estiver pronto, verá uma mensagem: "Your site is live at..."

6. **Aceda à Aplicação:**
   - A aplicação estará disponível em: **https://mariasambe.github.io/growup-kids/**

## Nota Importante

O ficheiro `index.html` foi criado para redirecionar automaticamente para `GrowUp_Kids.html`, por isso os utilizadores serão levados diretamente para a aplicação.

## Verificação

Após ativar, pode verificar se está a funcionar visitando:
- https://mariasambe.github.io/growup-kids/

Se houver algum problema, aguarde alguns minutos e tente novamente, pois o GitHub Pages pode demorar até 10 minutos para o primeiro deployment.
